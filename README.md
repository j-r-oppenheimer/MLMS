## 다운로드

[v1.0 APK](https://github.com/j-r-oppenheimer/MLMS/releases/tag/v1.0)

## 기능

- 시간표 조회 (주별, A/B반 필터)
- 수업 자료 다운로드
- 공지사항 조회
- 홈 화면 위젯
- 자동 로그인

## 빌드

- Android Studio에서 열고 Gradle Sync 후 실행
- minSdk 26 (Android 8.0)

## 사용 기술

Kotlin · MVVM · Hilt · Room · OkHttp · Jsoup · WorkManager
